# editor-cursor-skill

Edit Cursor Agent Skills in repo and/or global `~/.cursor/skills`. Edit once in repo, sync to global.

**What it does:** Edit skills; edit-once-then-sync workflow; determines repo vs global; keeps both in sync.

## Quick install

```bash
git clone -b editor-cursor-skill https://github.com/mariocosttaa/my-agent-skills.git && cp -r my-agent-skills/editor-cursor-skill ~/.cursor/skills/
```

→ [See all skills](https://github.com/mariocosttaa/my-agent-skills)
